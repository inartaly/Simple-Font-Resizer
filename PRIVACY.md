# Privacy Policy for Simple Font Resizer

**Last Updated: March 2026**

Simple Font Resizer is committed to providing a transparent and private experience for all users.

## 1. Data Collection
Simple Font Resizer **does not collect, store, or transmit** any personal data, browsing history, or user information. 

## 2. Permissions Used
The extension requires the following permissions to function:
* **Scripting:** Used only to apply font size changes to the webpage you are currently viewing.
* **Tabs:** Used to identify open tabs so the font size can be adjusted across your windows.

## 3. Third-Party Access
Because no data is collected, no data is ever shared with third parties. There are no tracking scripts, no ads, and no "Pro" version analytics.

## 4. Open Source
As an open-source project, you can verify the integrity of this extension by reviewing the source code at:
[https://github.com/inartaly/Simple-Font-Resizer](https://github.com/inartaly/Simple-Font-Resizer)
